## 疯狂星期四插件

### 要求
炸毛框架 > 3.0.2
php > 8.0

### 安装
```
./zhaomao plugin:install https://github.com/FishZe/zhamao_plugin_code_runner
```

### 使用

```
> code / 代码运行 / codeRunner / 运行代码 [可选语言]
> 请发送代码:
> [代码]
> 请发送输入:
> [stdin]
> [运行结果]
```
如:
```
> code python
> 请发送代码:
> print('hello world')
> 请发送输入:
> 0
> hello world
```
